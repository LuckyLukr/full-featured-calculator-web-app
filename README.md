# Vytvořte plnohodnotnou kalkulačku jako web app: hlavička s názvem aplikace a pře

Vytvořte plnohodnotnou kalkulačku jako web app: hlavička s názvem aplikace a přepínačem tématu (tmavé/světlé), elegantní rozhraní kalkulačky s displejem ukazujícím vstup a výsledek, kompletní mřížka tlačítek (číslice 0–9, operace +−×÷, vymazat, zpět, desetinná tečka, rovná se, závorky, procenta), paměťové funkce (M+, M−, MR, MC), podpora klávesnice přes JS, postranní panel s historií výpočtů, a patička s kredity. Použijte glassmorfismus nebo neumorfní design.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Tailwind CSS is preconfigured through Vite and loaded from `src/tailwind.css`.

Built with React + Vite. Deployed via Jean AI.
